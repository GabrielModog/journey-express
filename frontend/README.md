## Rodando o projeto

1. Instalação de dependências
```bash
yarn install
```

2. Execução
```bash
yarn dev
```

3. Rota padrão do projeto
```bash
http://localhost:3000
```

## Variáveis de Ambiente
```env
BACKEND_URL=http://localhost:3001/api
```
